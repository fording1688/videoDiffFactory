VISUAL VARIANT

处理视频

上传一个或多个素材，为每个视频独立生成视觉差异化版本。

多选视频会分别独立处理；输出会保存到上方文件夹，命名为 1.mp4、2.mp4、3.mp4 依次排列。字幕钩子开启后，会在视频前几秒叠加一条随机英文吸引文案。

URL DOWNLOAD

下载视频

粘贴 YouTube / Facebook / Instagram / TikTok 等平台的公开视频分享链接，下载后会保存到本地素材目录。

单条分享链接直接下载；主页、Reels 页、合集或频道链接请勾选批量下载，并设置数量上限。

MERGE

合并视频

选择两个或多个视频，系统会按选择顺序输出一个合并后的 MP4。

SPLIT

切分视频

选择一个大视频，按输入的随机秒数范围，从头到尾完整切分。

例如输入 50-56，会按 50 到 56 秒之间的随机长度，按顺序切完整个视频。最后一个片段可能略短。

SHORT DRAMA MVP

Short Drama Video Factory

Upload one raw drama video. The MVP detects high-emotion moments, cuts 15-35 second clips, and renders five hook angles per clip with metadata.

If a sidecar JSON/SRT/VTT transcript is present beside the source file on disk, it will be used. Otherwise the task uses local Whisper when installed, then falls back to evenly sampled candidate clips.
